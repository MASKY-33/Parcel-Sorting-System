# Parcel Sorting System
This system allows the user to enter a city name and instantly see which sorting hall it is linked to.
The program keeps running until the user types "stop", making it useful for continuous parcel‑sorting operations.

## Features
- Stores cities and their assigned sorting halls in a dictionary
- Lets the user search any city by name
- Checks if the city exists in the system
- Shows the correct sorting hall when found
- Gives a clear message if the city is not linked to any hall
- Runs in a continuous loop until the user types "stop"

## How to use
- Run the system and type the name of any city.
- The program will display the sorting hall assigned to that city.
- Type "stop" to exit the system.
